## 🚀 Local Installation and Development

If you would like to test the source codes of these extensions on your own computer or experiment with them, you can follow these steps:

1. Clone this repository to your computer: `git clone https://github.com/HuseyinOcak404/chrome-extensions.git`
2. Open Google Chrome and type `chrome://extensions/` into the address bar.
3. Enable the **Developer mode** option located in the top right corner.
4. Click the **Load unpacked** button in the top left and select the relevant extension folder in the repository.
